# DBA Administrative Task Coverage

This directory now includes operational scripts for:

- Oracle
- SQL Server
- PostgreSQL
- MySQL
- MongoDB
- Azure Cosmos DB
- Azure SQL Database
- Aurora PostgreSQL
- Aurora MySQL
- AWS RDS

## Task categories included per platform

- Monitoring / health checks
- Performance troubleshooting
- Maintenance operations
- Security and access audits
- HA/DR and replication checks
- Backup and restore runbooks

Use the scripts as starting points and tune thresholds per workload/SLA.
