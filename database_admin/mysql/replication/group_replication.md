# MySQL Group Replication (Active-Active)

## Overview
MySQL Group Replication enables multi-primary (active-active) setups.

## Modes
- Single-primary
- Multi-primary

## Key Concepts
- Quorum-based consistency
- Conflict detection
- GTID replication

## Basic Setup
- Enable GTID
- Configure group replication plugin
- Bootstrap first node

## Considerations
- Network latency
- Write conflicts
